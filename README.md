# Проект ИМ

После клонирования проекта из репозитория запустить:

```
./project init
```

эта команда установит все необходимые библиотеки для
работы приложения

```
./project dev
```

запустит dev сервер для работы приложения

по-умолчанию проект станет доступен по адресу

```
http://localhost:5000
```
